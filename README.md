# ИИ-сценарист для TikTok — Проектное предложение

Интерактивный сайт-презентация проекта для создания сценариев TikTok на основе референсов.

## Структура проекта

- `index.html` - главная страница с полным контентом
- `styles.css` - стили и анимации
- `script.js` - интерактивность и функционал
- `description.md` - описание работы сервиса
- `interface.md` - интерфейс и структура приложения
- `time-estimate.md` - оценка времени и стоимости

## Деплой на GitHub Pages

### Способ 1: Через настройки репозитория

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта в репозиторий
3. Перейдите в Settings → Pages
4. В разделе "Source" выберите branch `main` и папку `/ (root)`
5. Нажмите Save
6. Сайт будет доступен по адресу: `https://[ваш-username].github.io/[название-репозитория]`

### Способ 2: Через GitHub Actions (опционально)

Если хотите автоматический деплой при каждом push, можно настроить GitHub Actions workflow.

## Локальный запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js (с http-server)
npx http-server

# PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000` в браузере.
