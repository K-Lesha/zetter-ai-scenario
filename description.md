# Описание работы сервиса — ИИ-сценарист для TikTok

## Основной пользовательский флоу

### 1. Главная страница с таблицей референсов

Пользователь видит таблицу всех референс-видео с их статусами и сценариями. Каждая строка содержит:
- Референс-видео (название/ссылка)
- Выбранный суб-сценарий (dropdown с 3 вариантами)
- Статус (Черновик → В работе → Готово → Съемки → Опубликовано)
- Креатор (за кем закреплён сценарий)
- Действия (Просмотреть / Редактировать)

### 2. Добавление референса

Пользователь добавляет новый референс: загружает файл или вставляет ссылку на видео. Система автоматически / вручную:
- Расшифровывает видео в текст
- Определяет тип (нишевый/общий)
- Генерирует 3 варианта суб-сценариев

### 3. Выбор и доработка суб-сценария

Пользователь выбирает один из 3 суб-сценариев через dropdown в таблице. Затем уточняет параметры:
- Ниша
- Бренд/товар
- Тайминг видео (10–15 сек, 30 сек и т.д.)

### 4. Генерация финального сценария

ИИ адаптирует выбранный суб-сценарий под заданные параметры, создавая полноценный сценарий. Статус меняется на "В работе".

### 5. Доработка сценария

В режиме редактирования доступны действия:
- Дописать цепляющий хук
- Усилить интеграцию бренда
- Укоротить/удлинить сценарий
- Упростить/усложнить текст

После завершения статус меняется на "Готово".

## Командная работа

- Креаторы могут быть назначены на сценарии
- Фильтрация и сортировка по статусу, креатору, дате
- Экспорт готовых сценариев
